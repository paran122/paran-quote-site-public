## 1. 현재 완료된 작업

### 랜딩 페이지 (원페이지)
- 히어로 / 소개 / 포트폴리오 / 견적안내 / 프로세스 / 고객사 / 문의 섹션

### 포트폴리오 데이터 구조
- **DB**: Supabase `portfolios` + `portfolio_media` 테이블
- **저장소**: Supabase Storage `portfolio` 버킷 (WebP 최적화)
- **현황**: 15개 행사 / 540장 사진 업로드 완료

- 영상 있는 행사 → 카드에 영상 자동 재생
- 영상 없는 행사 → 현장사진 Ken Burns 효과 자동 순환

### 견적·문의 폼 → DB 저장
- 견적안내 "주문 요청하기" → Supabase `quotes` 테이블 저장 (QT-날짜-번호)
- 문의하기 폼 → 같은 테이블에 저장 (IQ-날짜-번호)
- 필수값 유효성 검증 (이름, 연락처, 이메일)

### SEO / 분석 (2026-02-26 설정 완료)
- Google Analytics 4 (G-323NGQ108L) 연동 → 방문자 트래픽·행동 분석 가능
- Google Search Console 등록 → 검색 노출 현황·색인 상태 모니터링 가능
- 네이버 서치어드바이저 등록 → 네이버 검색 노출 관리 가능
- OG 태그 최적화 (카카오톡·SNS 미리보기)
- JSON-LD 구조화 데이터
- sitemap.xml, robots.txt 자동 생성

### 보안
- middleware.ts로 관리자 경로 서버단 보호
- 로그인/관리자 페이지 제거 (불필요 공격면 축소)

## 4. 행사 포트폴리오 업로드 방법 (직원용)

행사 종료 후 포트폴리오를 홈페이지에 등록하는 방법입니다.
**Claude Code 터미널에서 아래 문구를 말하면 자동 처리됩니다.**

### 준비물
1. 행사 정보 MD 파일 (템플릿: `.claude/skills/event-portfolio-update/md-template.md`)
2. 현장 사진 파일 (JPG/PNG)
3. 시안물 이미지 (포스터, 배너 등)
4. 영상 파일 (있을 경우)

### 업로드 절차

| 단계 | Claude Code에 입력 | 자동 처리 내용 |
|------|-------------------|--------------|
| 1. 행사 등록 | "행사 등록해줘" + MD 파일 경로 | DB에 행사 기본정보 저장 |
| 2. 사진 추가 | "행사 사진 추가해줘" + 파일 | 자동 최적화(WebP) → Storage 업로드 → DB 연결 |
| 3. 영상 추가 | "행사 영상 추가해줘" + 파일 | 720p 압축 → 썸네일 생성 → 업로드 |
| 4. 후기 추가 | "행사 후기 넣어줘: 이름, 소속, 내용, 평점" | DB에 후기 저장 |

### 수정/삭제

| 작업 | 입력 |
|------|------|
| 행사 정보 수정 | "행사 수정해줘" |
| 사진 삭제 | "행사 사진 삭제해줘" |
| 사진 AI 보정 | "행사 사진 보정해줘" |
| 후기 수정 | "행사 후기 수정해줘" |

> **참고**: "행사" 또는 "포트폴리오" 키워드를 반드시 포함해야 합니다.
> 등록 후 `/work` 페이지에서 바로 확인 가능합니다.

## 5. 페이지 구성 및 데이터 흐름

```
[사용자 접속] → Netlify (정적 호스팅)
      │
      ▼
  Next.js 랜딩 페이지 (/)
      │
      ├─ 포트폴리오 섹션 ← Supabase DB (portfolios + portfolio_media)
      │                   ← Supabase Storage (portfolio 버킷, WebP)
      │
      ├─ 견적안내 → 주문 요청 폼 → Supabase DB (quotes 테이블)
      │
      ├─ 문의하기 → 문의 폼 → Supabase DB (quotes 테이블)
      │
      ├─ 고객사 후기 ← Supabase DB (event_reviews 테이블)
      │
      └─ 카카오톡 상담 → 카카오톡 채널 연결
```

### 주요 테이블 관계

| 테이블 | 용도 | 주요 컬럼 |
|--------|------|-----------|
| `portfolios` | 행사 기본정보 | title, client, event_type, tags, image_url |
| `portfolio_media` | 미디어 파일 | portfolio_id(FK), type, url, label, sort_order |
| `quotes` | 견적요청+문의 | quote_number, organization, contact_name, phone, email, event_type |
| `event_reviews` | 고객 후기 | portfolio_id(FK), reviewer_name, organization, content, rating |

도메인 `parancompany.co.kr`은 가비아에서 관리, DNS 연결 완료 (2026-02-26).

| 레코드 | 설정 | 상태 |
|--------|------|------|
| A 레코드 | Netlify IP | 연결 완료 |
| CNAME (www) | Netlify 도메인 | 연결 완료 |

| TXT 레코드 | SPF/DKIM 이메일 인증 | 유지 (변경 금지) |

> DNS 수정 시 MX/TXT 레코드는 회사 이메일(`@parancompany.co.kr`)에 영향을 주므로 절대 변경하지 말 것
