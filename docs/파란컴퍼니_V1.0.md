## 사업 영역

### 1. 행사 기획·운영 (핵심)

대규모 세미나·컨퍼런스·포럼·런칭행사·축제 등 행사의 기획부터 현장 운영까지 원스톱 수행.

**세부 서비스:**
| 서비스 | 설명 |
|--------|------|
| 세미나·컨퍼런스 운영 | 전문 주제 세미나, 학술 컨퍼런스 기획·진행 |
| 포럼 운영 | 국제·정책 포럼 기획·운영 (통역·하이브리드 포함) |
| 런칭행사 | 브랜드·사업 런칭 이벤트 기획·운영 |
| 축제·페스티벌 | 지역 축제, 문화 행사 종합 운영 |
| 교육·연수 프로그램 | 집체교육, 찾아가는 교육, 역량강화 프로그램 기획·운영 |
| 참가자 관리 | 사전등록 시스템, 현장 접수, 참석자 DB 관리 |
| 연사·강사 매칭 | 제휴 연사·강사 80+명 풀 기반 섭외·관리 |
| 결과보고서 작성 | 행사 성과 분석, 만족도 조사, 결과 보고서 납품 |

### 2. 행사 공간 디자인

행사 공간의 시각적 완성도를 높이는 디자인·제작 서비스.

**세부 서비스:**
| 서비스 | 설명 |
|--------|------|
| 로비·입구 연출 | 행사장 진입부 공간 디자인·시공 |
| 조형물 제작·설치 | 행사 테마 맞춤 조형물 디자인·제작·설치 |
| 체험부스·포토존 | 참가자 체험 공간, 포토존 기획·제작 |
| 키비주얼·사이니지 | 행사 핵심 비주얼 디자인, 안내 사인물 제작 |
| 무대·배경 디자인 | 무대 세트, 백드롭, LED 영상 디자인 |
| 현수막·배너 | 행사 홍보물, 현장 안내물 디자인·제작 |

### 3. 콘텐츠 제작

홍보·보고 영상, SNS 콘텐츠(카드뉴스·숏폼·웹툰) 등.

**세부 서비스:**
| 서비스 | 설명 |
|--------|------|
| 홍보·보고 영상 | 행사 홍보 영상, 결과 보고 영상 촬영·편집 |
| SNS 콘텐츠 | 카드뉴스, 숏폼 영상, 웹툰 등 소셜미디어 콘텐츠 |
| 온라인 교육 콘텐츠 | e-러닝 과정 기획·제작, 교육 영상 개발 |

## 특장점

- **대규모 행사 전문성**: 250+ 프로젝트 수행, 세미나·컨퍼런스·포럼·런칭행사·축제 기획·운영 경험, 90% 재계약률 달성
- **공간 디자인 역량**: 행사장 로비·입구 연출, 조형물 제작·설치, 체험부스·포토존 조성, 키비주얼·사이니지 디자인까지 원스톱
- **하이브리드 운영 역량**: 현장 행사와 YouTube·Zoom 생중계를 결합한 온·오프라인 동시 운영, 참가자 만족도 93점 이상 달성
- **맞춤형 솔루션**: 행사 규모·성격에 맞춘 식순 기획, 연사·공연자 섭외, 기관별 맞춤형 프로그램 구성
- **체계적 실행력**: 분야별 전문가 네트워크 보유, 고도화된 프로젝트 관리, 40여개 기관과의 파트너십 구축
- **전국 단위 운영**: 서울, 경기, 경남, 전남 등 전국 각지 현장 운영 대응 가능
