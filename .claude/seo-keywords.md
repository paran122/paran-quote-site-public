## 행사 제작물/디자인 키워드 (Google 검색량 기준)

블로그에서 행사 준비물·홍보물 관련 글을 쓸 때 활용.
"디자인" 키워드는 Google에서 검색량이 더 크고 경쟁이 낮으므로, 구글 SEO 목적 블로그에서는 "제작"보다 "디자인" 표현을 우선 사용.

## 활용처

| 위치 | 방법 |
|------|------|
| 블로그 제목 | title-guide.md 절차에 따라 관련 키워드 1~2개 포함 |
| 블로그 본문 H2 | 관련 키워드를 소제목에 자연스럽게 |
| 페이지 meta description | 해당 페이지 서비스와 매칭되는 키워드 포함 |
| 이미지 alt 텍스트 | 이미지 내용과 매칭되는 키워드 |
| 포트폴리오 설명 | 행사 유형에 맞는 키워드 (세미나, 컨퍼런스, 포럼 등) |

## 주의

- 이 리스트는 "행사 대행", "세미나", "케이터링" 등 **10개 검색어 기반 조사**이므로 모든 주제를 커버하지 않음
- 리스트에 없는 주제(큐시트, 결과보고서, 촬영 등)는 해당 주제로 별도 검색량 확인 필요
- 키워드를 억지로 넣지 말 것 — 내용과 자연스럽게 맞아야 SEO 효과가 있음
