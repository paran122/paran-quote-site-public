# 행사 포트폴리오 관리

## 필수 키워드

이 스킬 시스템은 **"행사" 또는 "포트폴리오"** 키워드가 포함된 경우에만 자동 트리거된다.
키워드 없이 범용 표현만 사용한 경우(예: "사진 추가해줘", "수정해줘") → 문맥 판단 후 확인 질문을 한다.

## 트리거 → 스킬 매핑

| 사용자 표현 | 스킬 | 동작 |
|------------|------|------|
| "행사 등록해줘" / "행사 올려줘" | `event-add` | 새 행사 등록 |
| "포트폴리오 추가해줘" | `event-add` | 새 행사 등록 |
| "행사 자료 올려줘" / "행사 DB에 넣어줘" | `event-add` | 새 행사 등록 |
| "행사 사진/영상 추가해줘" / "포트폴리오 사진/영상 추가해줘" | `event-photo-add` | 사진/영상 추가 |
| "행사 사진 빼줘" / "포트폴리오 사진 삭제해줘" | `event-photo-remove` | 사진 삭제 |
| "행사 후기 넣어줘" / "포트폴리오 후기 추가해줘" | `event-review-add` | 후기 추가 |
| "행사 후기 수정해줘" / "포트폴리오 후기 수정해줘" | `event-review-edit` | 후기 수정 |
| "행사 사진/영상 편집해줘" / "포트폴리오 사진/영상 편집해줘" | `event-photo-edit` | 사진/영상 편집 |
| "행사 사진 보정해줘" / "사진 퀄리티 올려줘" | `event-photo-edit` | 자동 보정 (화밸/노이즈/샤프닝/밝기) |
| "행사 영상 최적화해줘" / "포트폴리오 영상 최적화해줘" | `event-photo-edit` | 영상 최적화 (720p H.264) |
| "행사 수정해줘" / "포트폴리오 수정해줘" | `event-edit` | 기본 정보 수정 |

## 판단 기준

### 1단계: 필수 키워드 확인
- **"행사" 또는 "포트폴리오"가 포함** → 2단계로 진행
- **키워드 없음** → 3단계(문맥 판단)으로 진행

### 2단계: 동작 매핑
1. **MD 파일 제시 + "등록/올려/추가/넣어"** → `event-add`
2. **"사진/영상" + "추가"** → `event-photo-add`
3. **"사진" + "빼/삭제"** → `event-photo-remove`
4. **"사진/영상" + "편집/수정"** → `event-photo-edit`
5. **"사진" + "보정/퀄리티"** → `event-photo-edit` (자동 보정 모드)
6. **"영상" + "최적화"** → `event-photo-edit` (영상 최적화 모드)
7. **"후기" + "넣어/추가"** → `event-review-add`
8. **"후기" + "수정/바꿔"** → `event-review-edit`
9. **"수정/바꿔" (사진/후기/영상 없이)** → `event-edit`

### 3단계: 문맥 판단 (키워드 없는 경우)
"행사/포트폴리오" 키워드 없이 범용 표현만 사용한 경우, AskUserQuestion으로 확인:
- "행사 포트폴리오에 대한 작업인가요?"
  - 옵션 1: "네, 행사 포트폴리오 작업입니다" → 2단계로 진행
  - 옵션 2: "아니요, 다른 작업입니다" → 스킬 미실행
